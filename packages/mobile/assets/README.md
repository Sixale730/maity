# Assets

Este directorio contiene los recursos gráficos de la aplicación móvil.

## Archivos necesarios:

- **icon.png**: 1024x1024px - Ícono principal de la app
- **adaptive-icon.png**: 1024x1024px - Ícono adaptativo para Android
- **splash.png**: 1284x2778px - Pantalla de carga
- **favicon.png**: 48x48px - Ícono para web

Por ahora están usando imágenes placeholder. Para reemplazarlos:

1. Crea/obtén las imágenes con las dimensiones correctas
2. Asegúrate de que sean PNG con transparencia (excepto splash)
3. Usa colores de la paleta MAITY:
   - Primary: #8B5CF6
   - Secondary: #F97316
   - Tertiary: #06B6D4

## Herramientas recomendadas:

- https://www.figma.com (diseño)
- https://makeappicon.com (generador de íconos)
- https://hotpot.ai/app-icon-generator (generador con IA)
