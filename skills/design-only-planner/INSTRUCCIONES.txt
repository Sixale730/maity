# INSTRUCCIONES: Design-Only Planner

## Propósito
Esta habilidad sirve para planificar cambios visuales sin tocar código de forma impulsiva.

## Cómo usar
1.  **Solicitud**: Di "Planea el rediseño de [sección]".
2.  **Entrada (Input)**: Si tienes un boceto o lista de deseos, ponlo en `resources/input/requisitos.txt`.
3.  **Resultado (Output)**: Encontrarás el plan técnico en `resources/output/PLAN_DISENO.md`.

## Estructura de Carpetas
- `SKILL.md`: Lógica de planificación.
- `resources/input`: Documentos de requisitos o ideas.
- `resources/output`: Planes generados.
